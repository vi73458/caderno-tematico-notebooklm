# 📚 Caderno Temático com NotebookLM – DevOps na Prática

## 🎯 Contexto e Objetivos

Este projeto tem como objetivo utilizar Inteligência Artificial (NotebookLM) como ferramenta de aprendizagem ativa para estudar o tema **DevOps e Cultura de Automação de Software**.

A proposta é transformar fontes confiáveis em conhecimento estruturado, promovendo aprendizado crítico e organizado.

### 🎯 Objetivos do estudo:
- Compreender os fundamentos do DevOps
- Explorar práticas como CI/CD, automação e colaboração entre equipes
- Utilizar IA para sintetizar e organizar conhecimento
- Criar um material de estudo reutilizável (miniguia)

## 📖 Miniguia de Estudo (Resultado Final)

### 🧩 Resumo do Tema

DevOps é uma cultura e prática que une desenvolvimento e operações, promovendo automação, integração contínua e colaboração entre equipes.

Seu principal objetivo é acelerar o ciclo de desenvolvimento de software, tornando as entregas mais rápidas, seguras e confiáveis.

---

### 🔄 Antes e Depois do DevOps

**Antes:**
- Equipes separadas
- Processos lentos
- Falta de comunicação

**Depois:**
- Integração entre equipes
- Automação de processos
- Entregas contínuas e rápidas

---

### ⚙️ Principais conceitos

- **CI (Continuous Integration):** integração contínua de código  
- **CD (Continuous Delivery/Deployment):** entrega automatizada  
- **Containers:** ambientes isolados de execução (Docker)  
- **Orquestração:** gerenciamento de containers (Kubernetes)  

---

### 📌 Glossário

- **DevOps:** cultura de integração entre Dev e Ops  
- **Pipeline:** fluxo automatizado de entrega  
- **Container:** ambiente isolado para aplicações  
- **CI/CD:** automação de testes e deploy  
- **Cloud:** infraestrutura sob demanda  

---
